About Sonia Template

This design is a variation of the one on Solucija, a Croatian Website. It has nicely designed boxes to store various information and it's all packed up in soft colors. It has a small header box for the important links as well as the main navigation buttons. Remebmer that any CSS template could be easily implemented in sNews, a completly free PHP and MySQL driven news publishing tool. However, you are free to use this template in any way you want. No copyright notice or link back required!
